# BIIK-rasp-apk
nonoff приложуха для расписания с https://biik.ru/rasp/cg.htm. (мне некомфортно произвольное обновление расписания без уведомления.)

Кратко о приложении: Оно берет **именование вашей группы** переходит по ссылке что соотвествует вашей группы, копирует текстовые данные сайта с соотвествующих контейнеров и вставляет их на экран пользователя, предполагается автономное пользование и знание даже тер пар, что уже прошли.
Из приложения, пока только идея что будет реализовываться в последствии.
